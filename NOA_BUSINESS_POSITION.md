# NOA — AI에게 옷을 입혀서, 함께 동행하세요.

## 1. 핵심 인사이트

지금 AI 시장의 문제:

```
사용자: "Claude야, 의료 질문인데..."
Claude: (매번 처음부터 설명해야 함)
사용자: "아 그리고 난 의사야, 기본 설명 빼줘"
Claude: (다음 대화에선 또 까먹음)
```

**모든 AI가 매번 벌거벗은 상태로 나온다.** 사용자가 매번 옷을 입혀야 한다.

NOA의 답:

```
사용자: (medical.noa 입힘)
AI: (이미 의료 전문가 모드, 불필요한 면책 조항 최소화,
     근거 수준 자동 표기, 위험 신호 자동 분류)
사용자: "이 증상 조합이면?"
AI: (옷이 입혀져 있으니까 바로 핵심만)
```

---

## 2. 제품 정의

NOA는 "LLM용 정책 컴파일러 + 레이어드 프리셋 시스템 + 런타임 세션 관리자"다.

- **몸** = AI 모델 (Copilot, Claude, GPT, 로컬 모델)
- **옷** = `.noa` 정의 파일 (정체성, 정책, 톤, 행동 규칙)
- **악세사리** = 스킬, MCP, 도구 연동

핵심 가치:

- **이식성**: 하나의 `.noa`를 GPT, Claude, 로컬 모델로 옮긴다
- **설명 가능성**: 최종 규칙이 왜 그렇게 나왔는지 추적 가능
- **안전성**: 하위 레이어가 상위 안전 규칙을 무력화하지 못함
- **재사용성**: wardrobe, accessory, user preference를 모듈처럼 조합

## 4. 제품 라인업

### Free

- `.noa` 에디터 (VS Code Extension)
- 기본 옷 3벌 (medical, legal, creative)
- 레이어 2개까지 중첩
- Claude + GPT 어댑터
- 컴파일 & 내보내기

### Pro — 월 $19

- 무제한 레이어 중첩
- 전체 프리셋 옷장 (15+ 도메인)
- 엔진 전체 활성화 (HFCP, EH, HCRF, TLMH)
- Explain View (왜 이 규칙이 적용됐는지)
- Copilot Chat @noa 연동
- Local 모델 어댑터 (Ollama, LM Studio)
- 내 옷 마켓 등록 (판매 수익 80%)

### Team — 팀당 월 $99

- Pro 전체 기능
- 팀 유니폼 (전사 정책 강제)
- OCFP 기업 필터 활성화
- 감사 로그 (AegisLedger) 대시보드
- 관리자 Override / Seal 권한
- 팀원별 옷장 관리

### Enterprise — 연 계약

- Team 전체 기능
- 컴플라이언스 레이어 (금융/의료/법률 규제)
- SSO + RBAC + 감사 로그 SIEM 연동
- On-premise / Private Cloud 배포
- 커스텀 엔진 파라미터 튜닝
- 전담 지원

### Wardrobe Market — 수수료 20%

- 누구나 `.noa` 파일 제작 & 판매
- 리뷰 / 평점 / 다운로드 수
- 도메인별 카테고리
- 기업용 프라이빗 마켓

## 10. 해자 (Moat)

**네트워크 효과**
마켓에 `.noa` 파일이 많을수록 사용자 증가, 사용자가 많을수록 제작자 증가, 제작자가 많을수록 파일 증가.

**전환 비용**
팀이 유니폼 + 감사 로그 쓰기 시작하면 떠나기 어려움.

**데이터 자산**
"어떤 도메인에서 어떤 정책 조합이 효과적인지" 데이터 축적.

**엔진 IP**
HFCP, HCRF, EH, OCFP, TLMH, Sovereign, Aegis — 7개 엔진 특허 회피 설계 완료. 경쟁자가 따라하려면 이 전체 스택을 다시 만들어야 함.

**표준화**
`.noa`가 업계 표준 포맷이 되면 플랫폼 잠금.
